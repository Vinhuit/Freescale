TSS Libraries for IAR Embedded Workbench for ARM Version 6.50.2 or higher
=========================================================================

This folder contains platform-dependent TSS precompiled libraries for 
IAR Embedded Workbench for ARM Version 6.50.2 or higher. Please see the 
Reference Manual (TSSAPIRM) for more information about how to use this library.
The suffix of the library file name informs about the MCU platform 
which this file is intended for. 

Library support files are located in <install_path>/lib/shared folder
(<install_path> is typically C:\Program Files\Freescale\Freescale TSS x.xx)

Following versions of libraries are currently provided for IAR Embedded 
Workbench for ARM Version 6.50.2 or higher:
-------------------------------------------------------------------------
TSS_KXX_M0.a - ARM®Cortex™-M0 Kinetis platform
TSS_KXX_M4.a - ARM®Cortex™-M4 Kinetis platform
